AgriSense Dashboard

Real-time soil moisture monitoring system built for smart agriculture. An Arduino reads a soil sensor and sends data over USB to a Node.js server, which streams it live to a web dashboard. Shows current moisture level, a 12-hour history chart, field zone tracking, and triggers a red alert when moisture exceeds 80%.

  • Arduino
  • Node.js
  • HTML/CSS/JS
  • IoT
  • Serial API

Foody Buddy

Canteen ordering system backend built with Flask and PostgreSQL. Powers a platform where students can browse a menu, place food orders, make payments, and receive WhatsApp notifications. Includes a student frontend website and admin panel for managing canteen operations.
